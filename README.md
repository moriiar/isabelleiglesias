# Isabelle Iglesias

## 🗂️ Project Structure

- `index.html`: The main landing page.
- `assets/`: Directory containing images, stylesheets, and other static resources.
- `CNAME`: Optional file for custom domain configuration.

- HTML5
- CSS3
